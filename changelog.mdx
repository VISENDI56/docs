## Breaking changes

<Updates>
  <Update date="2025-12-25" type="breaking">
    **SovereignGuardrail Configuration**
    
    New configuration file required:
    - `config/sovereign_guardrail.yaml` must be created
    - Jurisdiction must be explicitly set
    - Data residency zones must be configured
    - Migration guide: See [Governance Configuration](/governance/overview)
  </Update>

  <Update date="2025-12-20" type="breaking">
    **Crypto Shredder Key Storage**
    
    Key storage path changed:
    - Old: `./governance_kernel/keys/`
    - New: `./keys/` (configurable)
    - Existing keys must be migrated
    - Migration script: `python scripts/migrate_keys.py`
  </Update>
</Updates>
