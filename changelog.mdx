## Migration Guides

### Upgrading from Beta

If you're upgrading from the beta version:

1. **Backup your data:**
   ```bash
   ./scripts/backup_data.sh
   ```

2. **Update dependencies:**
   ```bash
   pip install -r requirements.txt --upgrade
   ```

3. **Migrate configuration:**
   ```bash
   python scripts/migrate_config.py
   ```

4. **Validate fortress:**
   ```bash
   ./scripts/validate_fortress.sh
   ```

## Breaking Changes

### Version 1.0.0

- **SovereignGuardrail API:** Changed from function-based to class-based
  ```python
  # Old (Beta)
  validate_sovereignty(action, payload)
  
  # New (1.0.0)
  guardrail = SovereignGuardrail()
  guardrail.validate_action(action_type, payload, jurisdiction)
  ```

- **Crypto Shredder:** Retention policies now use enum
  ```python
  # Old (Beta)
  retention_days=180
  
  # New (1.0.0)
  retention_policy=RetentionPolicy.HOT
  ```

- **Golden Thread:** Verification score calculation updated
  ```python
  # Old (Beta)
  score = simple_match(cbs, emr)
  
  # New (1.0.0)
  score = quantum_entanglement_logic(cbs, emr, idsr)
  ```
