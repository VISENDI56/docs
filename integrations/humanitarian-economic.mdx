## Quick start

```python
from economic.humanitarian_economic import HumanitarianEconomicLayer

# Initialize layer
econ = HumanitarianEconomicLayer(economic_model="CEA")

# Calculate cost-effectiveness
cea = econ.calculate_cost_effectiveness({
    "intervention": "cholera_vaccination",
    "cost_usd": 100000,
    "dalys_averted": 500
})
print(f"Cost per DALY: ${cea.cost_per_daly}")

# Optimize resource allocation
allocation = econ.optimize_resource_allocation(
    budget=1000000,
    interventions=[
        {"name": "vaccination", "cost": 100000, "dalys": 500},
        {"name": "water_treatment", "cost": 50000, "dalys": 300}
    ]
)

# Assess economic impact
impact = econ.assess_economic_impact({
    "disease": "cholera",
    "cases": 1000,
    "deaths": 50,
    "duration_days": 90
})
print(f"Economic loss: ${impact.total_loss_usd}")
```

## DALY calculation

Disability-Adjusted Life Years (DALYs) = Years of Life Lost (YLL) + Years Lived with Disability (YLD)

```python
# Calculate DALYs
dalys = econ.calculate_dalys({
    "deaths": 50,
    "average_age_at_death": 35,
    "life_expectancy": 65,
    "disability_weight": 0.2,
    "duration_years": 0.25
})
```
