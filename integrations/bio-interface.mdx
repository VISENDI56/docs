## Overview

The Bio-Interface REST API enables mobile health applications to integrate with iLuminara-Core using the Golden Thread protocol for multi-source data fusion.

## Architecture

```
┌─────────────────────────────────────────────────────────┐
│              MOBILE HEALTH APPS                         │
│  - Community Health Volunteer (CHV) Apps               │
│  - Patient Self-Reporting Apps                         │
│  - Hospital EMR Systems                                │
│  - IoT Health Sensors                                  │
└─────────────────────────────────────────────────────────┘
                        │
                        │ HTTPS/TLS 1.3
                        ▼
┌─────────────────────────────────────────────────────────┐
│              BIO-INTERFACE REST API                     │
│  - /api/health/report (CBS signals)                    │
│  - /api/health/emr (EMR records)                       │
│  - /api/health/voice (Voice alerts)                    │
│  - /api/health/iot (Sensor data)                       │
└─────────────────────────────────────────────────────────┘
                        │
                        │ Validation
                        ▼
┌─────────────────────────────────────────────────────────┐
│              SOVEREIGNGUARDRAIL                         │
│  - Validates consent tokens                            │
│  - Enforces data residency                             │
│  - Checks PHI protection                               │
└─────────────────────────────────────────────────────────┘
                        │
                        │ Fusion
                        ▼
┌─────────────────────────────────────────────────────────┐
│              GOLDEN THREAD (IP-05)                      │
│  - Fuses CBS + EMR + IDSR signals                      │
│  - Calculates verification scores                      │
│  - Promotes to CONFIRMED if spatial/temporal match     │
└─────────────────────────────────────────────────────────┘
```

## Endpoints

### POST /api/health/report

Submit Community-Based Surveillance (CBS) report.

**Request:**

```json
{
  "reporter_id": "CHV_AMINA_HASSAN",
  "location": {
    "name": "Dadaab",
    "latitude": 0.0512,
    "longitude": 40.3129
  },
  "symptoms": ["diarrhea", "vomiting", "dehydration"],
  "severity": 9,
  "patient_count": 1,
  "timestamp": "2025-01-15T08:00:00Z",
  "consent_token": "CONSENT_TOKEN_12345",
  "metadata": {
    "age_group": "adult",
    "gender": "female"
  }
}
```

**Response:**

```json
{
  "status": "success",
  "signal_id": "CBS_SIG_789",
  "verification_status": "UNVERIFIED",
  "verification_score": 0.25,
  "message": "CBS signal recorded. Awaiting EMR confirmation.",
  "next_steps": [
    "Visit nearest health facility for diagnosis",
    "Continue monitoring symptoms"
  ]
}
```

### POST /api/health/emr

Submit Electronic Medical Record (EMR).

**Request:**

```json
{
  "facility_id": "DADAAB_CLINIC",
  "patient_id": "PAT_001",
  "location": {
    "name": "Dadaab",
    "latitude": 0.0515,
    "longitude": 40.3132
  },
  "diagnosis": "cholera",
  "symptoms": ["diarrhea", "vomiting"],
  "severity": 9,
  "timestamp": "2025-01-15T08:30:00Z",
  "consent_token": "CONSENT_TOKEN_12345",
  "lab_results": {
    "test_type": "stool_culture",
    "result": "positive_vibrio_cholerae"
  }
}
```

**Response:**

```json
{
  "status": "success",
  "record_id": "EMR_REC_456",
  "fusion_result": {
    "fused_record_id": "FUSED_123",
    "verification_status": "CONFIRMED",
    "verification_score": 1.0,
    "matched_signals": ["CBS_SIG_789"],
    "spatial_delta_km": 0.4,
    "temporal_delta_hours": 0.5
  },
  "alert_triggered": true,
  "alert_level": "CRITICAL"
}
```

### POST /api/health/voice

Process voice alert from CHV.

**Request:**

```bash
curl -X POST https://api.iluminara.health/api/health/voice \
  -H "Content-Type: audio/wav" \
  -H "X-Consent-Token: CONSENT_TOKEN_12345" \
  -H "X-Location: 0.0512,40.3129" \
  --data-binary @voice-alert.wav
```

**Response:**

```json
{
  "status": "success",
  "transcription": "Patient reporting severe watery diarrhea and vomiting",
  "language_detected": "swahili",
  "symptoms": ["diarrhea", "vomiting"],
  "severity": 9,
  "signal_id": "VOICE_SIG_321",
  "verification_status": "UNVERIFIED",
  "recommendations": [
    "IMMEDIATE: Suspected cholera. Start ORS immediately.",
    "Isolate patient to prevent spread",
    "Notify district health officer"
  ]
}
```

```json
  "location": {
```

```json
  }
}
```

## Golden Thread fusion

The Bio-Interface automatically triggers Golden Thread fusion:

```python
from core.fusion import GoldenThreadFusion, DataSignal, DataSource
from datetime import datetime

fusion = GoldenThreadFusion()

# CBS signal from mobile app
cbs_signal = DataSignal(
    source=DataSource.CBS,
    location="Dadaab",
    symptom="diarrhea",
    diagnosis=None,
    timestamp=datetime(2025, 1, 15, 8, 0),
    latitude=0.0512,
    longitude=40.3129,
    severity=9,
    metadata={"reporter_id": "CHV_AMINA"}
)

# EMR record from clinic
emr_signal = DataSignal(
    source=DataSource.EMR,
    location="Dadaab",
    symptom="diarrhea",
    diagnosis="cholera",
    timestamp=datetime(2025, 1, 15, 8, 30),
    latitude=0.0515,
    longitude=40.3132,
    severity=9,
    metadata={"facility_id": "DADAAB_CLINIC"}
)

# Fuse signals
fused = fusion.fuse_signals([cbs_signal, emr_signal], patient_id="PAT_001")

# Result:
# verification_score = 1.0 (CONFIRMED)
# spatial_delta_km = 0.4
# temporal_delta_hours = 0.5
```

## Mobile SDK

### Android (Kotlin)

```kotlin
import com.iluminara.sdk.BioInterface

class HealthReportActivity : AppCompatActivity() {
    private val bioInterface = BioInterface(
        apiKey = "YOUR_API_KEY",
        baseUrl = "https://api.iluminara.health"
    )
    
    fun submitReport() {
        val report = HealthReport(
            reporterId = "CHV_AMINA_HASSAN",
            location = Location("Dadaab", 0.0512, 40.3129),
            symptoms = listOf("diarrhea", "vomiting"),
            severity = 9,
            consentToken = "CONSENT_TOKEN_12345"
        )
        
        bioInterface.submitCBSReport(report) { result ->
            when (result) {
                is Success -> {
                    showToast("Report submitted: ${result.data.signalId}")
                }
                is Error -> {
                    showError(result.message)
                }
            }
        }
    }
}
```

### iOS (Swift)

```swift
import iLuminaraSDK

class HealthReportViewController: UIViewController {
    let bioInterface = BioInterface(
        apiKey: "YOUR_API_KEY",
        baseURL: "https://api.iluminara.health"
    )
    
    func submitReport() {
        let report = HealthReport(
            reporterId: "CHV_AMINA_HASSAN",
            location: Location(name: "Dadaab", latitude: 0.0512, longitude: 40.3129),
            symptoms: ["diarrhea", "vomiting"],
            severity: 9,
            consentToken: "CONSENT_TOKEN_12345"
        )
        
        bioInterface.submitCBSReport(report) { result in
            switch result {
            case .success(let response):
                self.showAlert("Report submitted: \\(response.signalId)")
            case .failure(let error):
                self.showError(error.localizedDescription)
            }
        }
    }
}
```

### React Native (JavaScript)

```javascript
import { BioInterface } from '@iluminara/react-native-sdk';

const bioInterface = new BioInterface({
  apiKey: 'YOUR_API_KEY',
  baseUrl: 'https://api.iluminara.health'
});

async function submitReport() {
  try {
    const response = await bioInterface.submitCBSReport({
      reporterId: 'CHV_AMINA_HASSAN',
      location: {
        name: 'Dadaab',
        latitude: 0.0512,
        longitude: 40.3129
      },
      symptoms: ['diarrhea', 'vomiting'],
      severity: 9,
      consentToken: 'CONSENT_TOKEN_12345'
    });
    
    console.log('Report submitted:', response.signalId);
  } catch (error) {
    console.error('Error:', error.message);
  }
}
```

## Authentication

### API key authentication

```bash
curl -X POST https://api.iluminara.health/api/health/report \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{...}'
```

### OAuth 2.0

```python
import requests

# Get access token
token_response = requests.post(
    'https://auth.iluminara.health/oauth/token',
    data={
        'grant_type': 'client_credentials',
        'client_id': 'YOUR_CLIENT_ID',
        'client_secret': 'YOUR_CLIENT_SECRET',
        'scope': 'health:write'
    }
)

access_token = token_response.json()['access_token']

# Use token for API requests
response = requests.post(
    'https://api.iluminara.health/api/health/report',
    headers={'Authorization': f'Bearer {access_token}'},
    json={...}
)
```

```json
{
  "webhook_url": "https://your-app.com/webhooks/iluminara",
  "events": [
    "signal.created",
    "signal.verified",
    "alert.triggered",
    "outbreak.detected"
  ],
  "secret": "WEBHOOK_SECRET_KEY"
}
```
