## Quick start

```python
from infrastructure.water_atm import WaterATMSovereignty

# Initialize ATM
atm = WaterATMSovereignty(
    atm_id="ATM-DADAAB-001",
    location={"lat": 0.0512, "lng": 40.3129}
)

# Authenticate user
auth_result = atm.authenticate_user({
    "biometric_type": "fingerprint",
    "biometric_data": fingerprint_data
})

# Dispense water
if auth_result.authenticated:
    atm.dispense_water(
        user_id=auth_result.user_id,
        liters=20
    )

# Monitor quality
quality = atm.monitor_water_quality()
print(f"pH: {quality.ph}, TDS: {quality.tds_ppm}")
```
