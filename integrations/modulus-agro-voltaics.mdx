## Quick start

```python
from sustainability.modulus_agro_voltaics import ModulusAgroVoltaics

# Initialize system
agro = ModulusAgroVoltaics(
    location={"lat": 0.0512, "lng": 40.3129},
    climate_zone="semi-arid"
)

# Optimize panel placement
result = agro.optimize_panel_placement({
    "field_dimensions": {"length_m": 100, "width_m": 50},
    "crop_type": "maize",
    "target_energy_kwh": 10000
})

print(f"Optimal configuration: {result.panel_layout}")
print(f"Expected crop yield: {result.crop_yield_kg}")
print(f"Water savings: {result.water_savings_liters}")
```
