## Overview

Tele-Justice Nodes provide access to justice for 200,000+ unrepresented refugees using secure Legal-LLM enclaves. The system combines confidential computing with specialized legal AI trained on Kenyan and International Refugee Law.

<Info>
  Refugees face complex legal issues (asylum status, disputes) but have no access to lawyers. Tele-Justice provides legal aid through privacy-hardened AI terminals protected by Confidential Computing.
</Info>

## Quick start

```python
from justice.tele_justice import TeleJusticeNode

# Initialize node
node = TeleJusticeNode(
    node_id="JUSTICE-DADAAB-001",
    jurisdiction="UNHCR_REFUGEE_LAW"
)

# Create case
case = node.create_case({
    "case_type": "family_reunification",
    "plaintiff": "REFUGEE-001",
    "description": "Family reunification request"
})

# Schedule hearing
node.schedule_hearing(
    case_id=case.case_id,
    participants=["REFUGEE-001", "LEGAL_AID_OFFICER"]
)

# Record verdict
node.record_verdict(
    case_id=case.case_id,
    verdict={"decision": "APPROVED", "reasoning": "..."}
)
```
