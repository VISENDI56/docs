## Overview

Blitzy System 2 Reasoning implements Kahneman's System 2 (slow, deliberate) thinking for high-stakes clinical decisions, providing multi-step reasoning, counterfactual analysis, and bias detection.

## Quick start

```python
from ai_reasoning.blitzy_system2 import BlitzySystem2Reasoning

# Initialize reasoning engine
reasoning = BlitzySystem2Reasoning(reasoning_depth=5)

# Analyze high-stakes decision
result = reasoning.analyze_decision({
    "decision_type": "treatment_protocol",
    "patient_data": {"age": 35, "symptoms": ["fever", "cough"]},
    "available_treatments": ["treatment_a", "treatment_b"],
    "constraints": {"budget": 1000, "time_hours": 24}
})

print(f"Recommended action: {result.recommendation}")
print(f"Reasoning chain: {result.reasoning_chain}")
print(f"Detected biases: {result.biases_detected}")
print(f"Confidence: {result.confidence}")
```

## Bias detection

<AccordionGroup>
  <Accordion title="Confirmation bias">
    Tendency to favor information confirming existing beliefs
  </Accordion>
  <Accordion title="Availability heuristic">
    Overweighting recent or memorable events
  </Accordion>
  <Accordion title="Anchoring bias">
    Over-reliance on first piece of information
  </Accordion>
  <Accordion title="Sunk cost fallacy">
    Continuing investment due to past costs
  </Accordion>
</AccordionGroup>
