## Prerequisites

Install required dependencies:

```bash
pip install streamlit pandas pydeck numpy Flask flask-cors
```

## Launch the war room demo

### Option 1: Auto-launch (recommended)

```bash
chmod +x launch_war_room.sh
./launch_war_room.sh
```

This single command will:
- Generate fresh outbreak simulation data
- Start all three dashboards
- Automatically open all dashboards in Chrome

### Option 2: Manual launch

```bash
# Generate outbreak data
python edge_node/frenasa_engine/simulate_outbreak.py

# Launch dashboard
streamlit run dashboard.py
```

Access the dashboard at `http://localhost:8501`

## Test the API

### Start the API service

```bash
python api_service.py
```

The service starts on `http://localhost:8080`

### Test voice processing

```bash
# Generate test audio
python generate_test_audio.py

# Process voice alert
curl -X POST http://localhost:8080/process-voice \
  -H "Content-Type: audio/wav" \
  --data-binary @swahili-symptom.wav
```

### Test outbreak prediction

```bash
curl -X POST http://localhost:8080/predict \
  -H "Content-Type: application/json" \
  -d '{
    "location": {"lat": 0.4221, "lng": 40.2255},
    "symptoms": ["diarrhea", "vomiting"]
  }'
```
