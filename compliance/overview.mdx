## Automated evidence generation

### Generate compliance bundle

```bash
python scripts/generate_compliance_bundle.py
```

**Output files:**
- `encryption_proofs.json` - Encryption-at-rest evidence
- `iam_least_privilege_logs.json` - Access control evidence
- `sovereign_guardrail_stats.json` - Sovereignty enforcement stats
- `nist_sp_800_53_mapping.json` - NIST control mappings
- `iso_27001_annex_a_mapping.json` - ISO control mappings
- `soc2_trust_criteria_mapping.json` - SOC 2 criteria mappings
- `EXECUTIVE_SUMMARY.md` - Executive summary for procurement

## NIST SP 800-53 controls

| Control | Name | iLuminara Component |
|---------|------|---------------------|
| AC-2 | Account Management | IAM, SovereignGuardrail |
| AC-6 | Least Privilege | IAM, RBAC |
| AU-2 | Audit Events | Audit Trail, Tamper-Proof Logging |
| AU-9 | Protection of Audit Information | Tamper-Proof Audit, Cloud Spanner |
| SC-13 | Cryptographic Protection | AES-256-GCM, Crypto Shredder |
| SC-28 | Protection of Information at Rest | Crypto Shredder, Encryption at Rest |

## Continuous compliance

### Real-time monitoring

```python
from governance_kernel.telemetry_bridge import TelemetryBridge

bridge = TelemetryBridge(audit_trail_client)

# Map events to controls
mappings = bridge.map_event_to_controls(event)

# Generate weekly report
report = bridge.generate_weekly_report()
```

### Automated reporting

- **Daily:** Security scan results (CodeQL, Gitleaks)
- **Weekly:** Compliance summary PDF
- **Monthly:** Full compliance bundle
- **Quarterly:** Executive review
