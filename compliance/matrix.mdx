| Framework | Restriction | Mechanism | iLuminara Enforcement |
|-----------|-------------|-----------|----------------------|
| **GDPR** | Art. 44-50 | Adequacy, SCCs, BCRs | Block non-EU transfers |
| **KDPA** | ยง37 | Commissioner approval | Block non-Kenya transfers |
| **HIPAA** | ยง164.312(e) | BAA required | Encrypted transmission |
| **PIPEDA** | - | Substantially similar | Approved jurisdictions |
| **POPIA** | ยง72 | Adequate protection | Block non-SA transfers |
| **CCPA** | - | No restriction | - |

```python

    action_type='Data_Transfer',
    payload={
        'data_type': 'PHI',
        'source_jurisdiction': 'KDPA_KE',
        'destination': 'US_Cloud'
    },
    jurisdiction='KDPA_KE'
)

```

## Retention requirements

| Framework | Minimum | Maximum | iLuminara Policy |
|-----------|---------|---------|------------------|
| **GDPR** | - | Necessary period | 180 days (HOT) |
| **KDPA** | - | Necessary period | 180 days (HOT) |
| **HIPAA** | 6 years | - | 1825 days (COLD) |
| **PIPEDA** | - | Necessary period | 365 days (WARM) |
| **POPIA** | - | Necessary period | 180 days (HOT) |
| **CCPA** | - | Necessary period | 180 days (HOT) |

```python
from governance_kernel.crypto_shredder import RetentionPolicy

# Framework-specific retention
retention_map = {
    'GDPR_EU': RetentionPolicy.HOT,      # 180 days
    'KDPA_KE': RetentionPolicy.HOT,      # 180 days
    'HIPAA_US': RetentionPolicy.COLD,    # 1825 days (5 years)
    'PIPEDA_CA': RetentionPolicy.WARM,   # 365 days
    'POPIA_ZA': RetentionPolicy.HOT,     # 180 days
    'CCPA_US': RetentionPolicy.HOT,      # 180 days
}
```

## Compliance validation

### Manual audit checklist

<Steps>
  <Step title="Data inventory">

  </Step>
