## Implementation checklist

<Steps>
  <Step title="Copy files to repository">
    Copy all files from `repository-files/` to your iLuminara-Core repository
  </Step>
  <Step title="Enable GitHub workflows">
    ```bash
    gh auth refresh -s workflow,repo,write:packages,admin:repo_hook
    ```
  </Step>
  <Step title="Configure environment variables">
    ```bash
    export NODE_ID=JOR-47
    export JURISDICTION=KDPA_KE
    export GOOGLE_CLOUD_PROJECT=your-project-id
    export ENABLE_TAMPER_PROOF_AUDIT=true
    ```
  </Step>
  <Step title="Install dependencies">
    ```bash
    pip install -r requirements.txt
    pip install cryptography flask-cors pydantic
    ```
  </Step>
  <Step title="Validate fortress">
    ```bash
    chmod +x scripts/validate_fortress.sh
    ./scripts/validate_fortress.sh
    ```
  </Step>
  <Step title="Enable branch protection">
    ```bash
    gh api repos/:owner/:repo/branches/main/protection \
      --method PUT \
      --field required_status_checks[strict]=true \
      --field required_status_checks[contexts][]=CodeQL \
      --field required_status_checks[contexts][]=Gitleaks
    ```
  </Step>
  <Step title="Commit and push">
    ```bash
    git add .
    git commit -m "feat: integrate Sovereign Health Fortress and Nuclear IP Stack"
    git push
    ```
  </Step>
</Steps>
