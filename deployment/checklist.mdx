## Manual validation

If you prefer manual validation, check each component:

### Security workflows

```bash
ls -la .github/workflows/
# Should show: codeql.yml, gitleaks.yml
```

### Governance kernel

```bash
ls -la governance_kernel/
# Should show: vector_ledger.py, crypto_shredder.py, ethical_engine.py
```

### Configuration

```bash
ls -la config/
# Should show: sovereign_guardrail.yaml
```

### Python packages

```bash
pip list | grep -E "cryptography|flask|streamlit|google-cloud"
```

### Environment variables

```bash
echo $NODE_ID
echo $JURISDICTION
echo $GOOGLE_CLOUD_PROJECT
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Missing dependencies">
    ```bash
    pip install -r requirements.txt
    ```
  </Accordion>
  <Accordion title="Environment variables not set">
    ```bash
    export NODE_ID=JOR-47
    export JURISDICTION=KDPA_KE
    export GOOGLE_CLOUD_PROJECT=your-project-id
    ```
  </Accordion>
  <Accordion title="Permission denied on scripts">
    ```bash
    chmod +x scripts/*.sh
    ```
  </Accordion>
  <Accordion title="GitHub workflows not found">
    Ensure you've copied files from `repository-files/` to your repository
  </Accordion>
</AccordionGroup>
