## Overview

The **Bio-Interface API** provides REST endpoints for mobile health applications to integrate with iLuminara's Golden Thread protocol, enabling zero-friction data collection from 14M+ African mobile nodes.

## Base URL

```
https://api.iluminara.health/v1
```

For local development:
```
http://localhost:8080/v1
```

## Authentication

All API requests require authentication using JWT tokens:

```bash
curl -X POST https://api.iluminara.health/v1/auth/token \
  -H "Content-Type: application/json" \
  -d '{
    "client_id": "your_client_id",
    "client_secret": "your_client_secret"
  }'
```

**Response:**

```json
{
  "access_token": "eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9...",
  "token_type": "Bearer",
  "expires_in": 3600
}
```

Use the token in subsequent requests:

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.iluminara.health/v1/health-check
```

## Core endpoints

### Health check

```http
GET /v1/health-check
```

Check API service health and metadata.

**Response:**

```json
{
  "status": "healthy",
  "version": "1.0.0",
  "fortress_status": "OPERATIONAL",
  "sovereignty_zone": "africa-south1",
  "timestamp": "2025-12-25T10:00:00.000Z"
}
```

### Submit health report

```http
POST /v1/reports/submit
```

Submit health report from Community Health Volunteer (CHV) or mobile app.

**Request:**

```json

  "patient_id": "PAT_12345",
  "location": {
    "lat": 0.0512,
    "lng": 40.3129,
    "name": "Dadaab Refugee Camp"
  },
  "symptoms": [
    {
      "name": "fever",
      "severity": 8,
      "duration_hours": 24
    },
    {
      "name": "diarrhea",
      "severity": 9,
      "duration_hours": 12
    }
  ],
  "vital_signs": {
    "temperature": 38.5,
    "heart_rate": 95,
    "blood_pressure": "120/80"
  },
  "consent_token": "VALID_TOKEN",
  "source": "CHV_MOBILE_APP",
  "timestamp": "2025-12-25T10:00:00.000Z"
}
```

**Response:**

```json

    "jurisdiction": "KDPA_KE",
    "validated": true
  }
}
```

### Voice alert processing

```http
POST /v1/voice/process
```

Process voice alert from CHV (Swahili/English).

**Request:**

```bash
curl -X POST https://api.iluminara.health/v1/voice/process \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: audio/wav" \
  --data-binary @swahili-symptom.wav
```

**Response:**

```json
{
  "status": "success",
  "transcription": "Patient reporting severe watery diarrhea and vomiting",
  "language_detected": "swahili",
  "symptoms": ["diarrhea", "vomiting", "dehydration"],
  "severity": 9,
  "alert_level": "CRITICAL",
  "report_id": "RPT_790",
  "golden_thread": {
    "verification_score": 0.75,
    "status": "PROBABLE"
  }
}
```

## Mobile SDK

### Android (Kotlin)

```kotlin
import com.iluminara.sdk.ILuminaraClient
import com.iluminara.sdk.models.HealthReport

// Initialize client
val client = ILuminaraClient(
    apiKey = "your_api_key",
    baseUrl = "https://api.iluminara.health/v1"
)

// Submit health report
val report = HealthReport(
    patientId = "PAT_12345",
    location = Location(0.0512, 40.3129),
    symptoms = listOf(
        Symptom("fever", severity = 8, durationHours = 24),
        Symptom("diarrhea", severity = 9, durationHours = 12)
    ),
    vitalSigns = VitalSigns(
        temperature = 38.5,
        heartRate = 95
    ),
    consentToken = "VALID_TOKEN"
)

client.submitReport(report) { result ->
    when (result) {
        is Success -> {
            println("Report submitted: ${result.data.reportId}")
            println("Alert level: ${result.data.alertLevel}")
        }
        is Error -> {
            println("Error: ${result.message}")
        }
    }
}
```

## Offline support

The Bio-Interface API supports offline-first mobile apps with intelligent sync:

### Queue reports offline

```javascript
// Queue report when offline
client.queueReport(report);

// Auto-sync when connectivity returns
client.on('online', () => {
  client.syncQueuedReports()
    .then(results => {
      console.log(`Synced ${results.length} reports`);
    });
});
```

### Conflict resolution

```javascript
// Handle sync conflicts
client.on('conflict', (conflict) => {
  // Server version vs local version
  const resolution = resolveConflict(
    conflict.serverVersion,
    conflict.localVersion
  );
  
  client.resolveConflict(conflict.id, resolution);
});
```

## Error codes

| Code | HTTP Status | Description |
|------|-------------|-------------|
| `invalid_token` | 401 | Invalid or expired JWT token |
| `missing_consent` | 403 | Patient consent required |
| `sovereignty_violation` | 403 | Action violates data sovereignty |
| `rate_limit_exceeded` | 429 | Too many requests |
| `invalid_location` | 400 | Invalid GPS coordinates |
| `processing_failed` | 500 | Internal processing error |

## Webhooks

Subscribe to real-time events:

```http
POST /v1/webhooks/subscribe
```

**Request:**

```json
{
  "url": "https://your-app.com/webhooks/iluminara",
  "events": [
    "report.submitted",
    "outbreak.detected",
    "alert.critical"
  ],
  "secret": "your_webhook_secret"
}
```

**Webhook payload:**

```json
{
  "event": "outbreak.detected",
  "timestamp": "2025-12-25T10:00:00.000Z",
  "data": {
    "location": "Dadaab",
    "disease": "cholera",
    "z_score": 8.3,
    "estimated_cases": 145
  },
  "signature": "sha256=..."
}
```
