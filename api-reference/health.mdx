## Endpoint

```
GET /health
```

## Description

Returns the health status of the iLuminara API service, including version information, uptime, and system metadata.

## Request

No parameters required.

```bash
curl http://localhost:8080/health
```

## Response

### Success (200 OK)

```json
{
  "status": "healthy",
  "version": "1.0.0",
  "service": "iLuminara-Core API",
  "timestamp": "2025-12-24T11:00:00.000Z",
  "uptime_seconds": 3600,
  "node_id": "JOR-47",
  "jurisdiction": "KDPA_KE",
  "fortress_status": "OPERATIONAL",
  "components": {
    "governance_kernel": "operational",
    "crypto_shredder": "operational",
    "golden_thread": "operational",
    "ai_agents": "operational"
  },
  "nuclear_ip_stack": {
    "IP_02_CRYPTO_SHREDDER": "active",
    "IP_03_ACORN_PROTOCOL": "requires_hardware",
    "IP_04_SILENT_FLUX": "active",
    "IP_05_GOLDEN_THREAD": "active",
    "IP_06_5DM_BRIDGE": "requires_mobile_network"
  }
}
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `status` | string | Health status ("healthy" or "unhealthy") |
| `version` | string | API version |
| `service` | string | Service name |
| `timestamp` | string | Current server timestamp (ISO 8601) |
| `uptime_seconds` | integer | Service uptime in seconds |
| `node_id` | string | Edge node identifier |
| `jurisdiction` | string | Primary jurisdiction |
| `fortress_status` | string | Overall Fortress status |
| `components` | object | Status of individual components |
| `nuclear_ip_stack` | object | Status of Nuclear IP protocols |

## Component status values

| Status | Description |
|--------|-------------|
| `operational` | Component is fully operational |
| `degraded` | Component is operational but with reduced performance |
| `offline` | Component is not available |
| `requires_hardware` | Component requires hardware integration |
| `requires_mobile_network` | Component requires mobile network integration |

```yaml
livenessProbe:
  httpGet:
    path: /health
    port: 8080
  initialDelaySeconds: 30
  periodSeconds: 10
```

```yaml

  httpGet:
    path: /health
    port: 8080
  initialDelaySeconds: 5
  periodSeconds: 5
```

```bash

HEALTH_URL="http://localhost:8080/health"
```

### Python health check

```python
import requests
import time

def check_health():
    try:
        response = requests.get('http://localhost:8080/health', timeout=5)
        data = response.json()
        
        if data['status'] == 'healthy':
            print(f"‚úÖ Service healthy - Uptime: {data['uptime_seconds']}s")
            return True
        else:
            print(f"‚ö†Ô∏è Service unhealthy: {data['status']}")
            return False
    
    except Exception as e:
        print(f"‚ùå Health check failed: {e}")
        return False

# Continuous monitoring
while True:
    check_health()
    time.sleep(60)
```
