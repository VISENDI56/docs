## Overview

iLuminara-Core is designed for **offline-first** operation, ensuring critical health surveillance continues even without internet connectivity.

## Offline capabilities

<CardGroup cols={2}>
  <Card title="Local processing" icon="microchip">
    All critical functions work without network
  </Card>
  <Card title="Data buffering" icon="database">
    SQLite buffer stores events until sync
  </Card>
  <Card title="AI inference" icon="brain">
    Edge AI models run locally
  </Card>
  <Card title="Auto-sync" icon="arrows-rotate">
    Automatic sync when connectivity returns
  </Card>
</CardGroup>

## Offline workflow

<Steps>
  <Step title="Detect offline status">
    System automatically detects network loss
  </Step>
  <Step title="Switch to local mode">
    All operations use local resources
  </Step>
  <Step title="Buffer events">
    Events stored in SQLite buffer
  </Step>
  <Step title="Continue operations">
    CHVs continue data collection
  </Step>
  <Step title="Detect connectivity">
    System detects network return
  </Step>
  <Step title="Auto-sync">
    Buffered events sync to Golden Thread
  </Step>
</Steps>

## Configuration

```bash
# Enable offline mode
export ENABLE_OFFLINE_MODE=true
export SYNC_INTERVAL_SECONDS=300

# Buffer configuration
export BUFFER_MAX_SIZE_MB=500
export BUFFER_MAX_EVENTS=10000
```
