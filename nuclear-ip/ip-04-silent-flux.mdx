## Overview

Silent Flux (IP-04) implements **anxiety-regulated AI output** - the system monitors operator stress levels and automatically reduces information verbosity to prevent cognitive overload during emergencies.

<Card
  title="Philosophy"
  icon="brain"
>
  "When anxiety rises, the signal must quiet. Clarity emerges from silence."
</Card>

## The problem

During health emergencies, AI systems generate overwhelming amounts of information:
- **Alert fatigue** - Too many notifications cause operators to ignore critical alerts
- **Cognitive overload** - Information flood prevents effective decision-making
- **Panic amplification** - Excessive data increases operator anxiety
- **Decision paralysis** - Too many options prevent action

Traditional AI systems have no awareness of operator state and continue generating output regardless of stress levels.

## The solution

Silent Flux monitors operator anxiety through multiple signals and dynamically adjusts AI output:

<Steps>
  <Step title="Anxiety detection">
    Monitor heart rate variability, typing speed, mouse movement patterns
  </Step>
  <Step title="Output regulation">
    Reduce verbosity, filter non-critical alerts, simplify recommendations
  </Step>
  <Step title="Adaptive interface">
    Simplify UI, increase font size, reduce color complexity
  </Step>
  <Step title="Cognitive support">
    Provide decision trees, highlight critical actions, suppress details
  </Step>
</Steps>

### Behavioral signals

```python
class BehavioralAnxietyDetector:
    """Detect anxiety from typing and mouse behavior"""
    
    def __init__(self):
        self.typing_buffer = []
        self.mouse_buffer = []
    
    def analyze_typing_speed(self, keystroke_events):
        """Rapid, erratic typing indicates anxiety"""
        intervals = [
            events[i+1].timestamp - events[i].timestamp
            for i in range(len(events)-1)
        ]
        
        # High variance = anxiety
        variance = np.var(intervals)
        anxiety_score = min(1.0, variance / 100)
        
        return anxiety_score
    
    def analyze_mouse_movement(self, mouse_events):
        """Erratic mouse movement indicates anxiety"""
        # Calculate path length
        path_length = sum([
            np.linalg.norm([
                events[i+1].x - events[i].x,
                events[i+1].y - events[i].y
            ])
            for i in range(len(events)-1)
        ])
        
        # Calculate straightness (direct distance / path length)
        direct_distance = np.linalg.norm([
            events[-1].x - events[0].x,
            events[-1].y - events[0].y
        ])
        
        straightness = direct_distance / path_length if path_length > 0 else 1.0
        
        # Low straightness = anxiety
        anxiety_score = 1.0 - straightness
        
        return anxiety_score
```

## Output regulation

### Verbosity control

```python
class VerbosityController:
    """Adjust AI output verbosity based on anxiety"""
    
    def regulate_output(self, ai_response, anxiety_score):
        """
        Reduce verbosity when anxiety is high.
        
        anxiety_score:
          0.0-0.3: Full detail
          0.3-0.6: Moderate detail
          0.6-0.8: Minimal detail
          0.8-1.0: Critical only
        """
        if anxiety_score < 0.3:
            # Low anxiety - full detail
            return ai_response
        
        elif anxiety_score < 0.6:
            # Moderate anxiety - summarize
            return self._summarize(ai_response, max_sentences=5)
        
        elif anxiety_score < 0.8:
            # High anxiety - bullet points only
            return self._extract_key_points(ai_response, max_points=3)
        
        else:
            # Critical anxiety - single action only
            return self._extract_critical_action(ai_response)
```

### Alert filtering

```python
class AlertFilter:
    """Filter non-critical alerts when anxiety is high"""
    
    def filter_alerts(self, alerts, anxiety_score):
        """
        Filter alerts based on anxiety level.
        
        anxiety_score:
          0.0-0.3: Show all alerts
          0.3-0.6: Show HIGH and CRITICAL only
          0.6-0.8: Show CRITICAL only
          0.8-1.0: Suppress all alerts (operator overloaded)
        """
        if anxiety_score < 0.3:
            return alerts
        
        elif anxiety_score < 0.6:
            return [a for a in alerts if a.level in ['HIGH', 'CRITICAL']]
        
        elif anxiety_score < 0.8:
            return [a for a in alerts if a.level == 'CRITICAL']
        
        else:
            # Operator is overloaded - suppress all alerts
            # Log for later review
            self._log_suppressed_alerts(alerts)
            return []
```

### UI adaptation

```python
class UIAdapter:
    """Adapt UI based on anxiety level"""
    
    def adapt_interface(self, anxiety_score):
        """
        Simplify UI when anxiety is high.
        
        Adaptations:
          - Increase font size
          - Reduce color complexity
          - Hide non-essential elements
          - Highlight critical actions
        """
        config = {
            "font_size": self._calculate_font_size(anxiety_score),
            "color_scheme": self._select_color_scheme(anxiety_score),
            "show_details": anxiety_score < 0.6,
            "show_sidebar": anxiety_score < 0.4,
            "highlight_critical": anxiety_score > 0.6
        }
        
        return config
    
    def _calculate_font_size(self, anxiety_score):
        """Increase font size with anxiety"""
        base_size = 14
        max_increase = 6
        return base_size + int(anxiety_score * max_increase)
    
    def _select_color_scheme(self, anxiety_score):
        """Simplify colors with anxiety"""
        if anxiety_score < 0.3:
            return "full_color"
        elif anxiety_score < 0.6:
            return "reduced_color"
        else:
            return "high_contrast"  # Black and white with red highlights
```
