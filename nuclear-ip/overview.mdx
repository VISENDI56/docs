## Why "Nuclear"?

The term "Nuclear IP" signifies:

1. **Atomic-level innovation** - These are fundamental building blocks, not surface features
2. **Irreducible complexity** - Each innovation cannot be simplified without losing its essence
3. **Chain reaction potential** - One innovation enables the next
4. **Defensive moat** - Extremely difficult to replicate
5. **Sovereign power** - Gives nations control over their health data destiny

## Quick start

### Enable Crypto Shredder

```python
from governance_kernel.crypto_shredder import CryptoShredder, RetentionPolicy

shredder = CryptoShredder(sovereignty_zone=SovereigntyZone.KENYA)

# Encrypt with ephemeral key
encrypted_data, key_id = shredder.encrypt_with_ephemeral_key(
    data=patient_record,
    retention_policy=RetentionPolicy.HOT
)

# After retention period, shred key
shredder.shred_key(key_id)
```

### Enable Golden Thread

```python
from edge_node.sync_protocol.golden_thread import GoldenThread

gt = GoldenThread()

# Fuse CBS and EMR signals
fused = gt.fuse_data_streams(
    cbs_signal={'location': 'Dadaab', 'symptom': 'fever'},
    emr_record={'location': 'Dadaab', 'diagnosis': 'malaria'},
    patient_id='PAT_001'
)

print(f"Verification Score: {fused.verification_score}")
```

### Enable Acorn Protocol (requires hardware)

```python
from nuclear_ip.acorn_protocol import AcornAuthenticator

acorn = AcornAuthenticator(
    tpm_device="/dev/tpm0",
    authorized_zones=[{"lat": 0.0512, "lng": 40.3129, "radius_m": 100}]
)

# Authenticate high-risk operation
if acorn.authenticate_high_risk_operation():
    # Proceed with operation
    pass
```

### Enable Silent Flux (requires monitoring)

```python
from nuclear_ip.silent_flux import SilentFluxEngine

silent_flux = SilentFluxEngine()

# Get operator anxiety score
anxiety_score = silent_flux.get_current_anxiety_score()

# Regulate AI output
regulated_output = silent_flux.regulate_output(ai_response, anxiety_score)
```
