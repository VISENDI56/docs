## Step 24: Silent Flux (IP-04)

Anxiety-regulated AI output that prevents cognitive overload during emergencies.

### Stress monitoring

Silent Flux calculates operator stress from:
- **Keystroke cadence** - Optimal: 3-5 keys/second
- **Error rate** - Errors per action
- **Response time** - Optimal: 1-3 seconds
- **UI interaction rate** - Optimal: 10-20 per minute

### Output modes

| Stress Level | Output Mode | Max Tokens | Behavior |
|--------------|-------------|------------|----------|
| CALM (0.0-0.3) | FULL | 2000 | All details, explanations, context |
| FOCUSED (0.3-0.5) | STANDARD | 1000 | Normal operation |
| ELEVATED (0.5-0.7) | CONCISE | 500 | Reduced verbosity |
| HIGH (0.7-0.85) | ESSENTIAL | 200 | Critical information only |
| CRITICAL (0.85-1.0) | EMERGENCY | 100 | Absolute minimum, action-oriented |

### Usage

```python
from governance_kernel.silent_flux import get_silent_flux

flux = get_silent_flux()

# Record operator activity
flux.record_keystroke()
flux.record_action()
flux.record_response_time(1.5)

# Get current status
status = flux.get_status()
print(f\"Stress Level: {status['stress_level']}\")
print(f\"Output Mode: {status['output_mode']}\")

# Regulate AI output
ai_output = \"This is a detailed explanation of the outbreak situation...\"
regulated = flux.regulate_output(ai_output)
```

## Step 25: Acorn Protocol (IP-03)

Somatic-keyed secret injection using posture + location + stillness as cryptographic authentication.

### Biometric authentication

Acorn Protocol prevents \"panic access\" by requiring:
- **Authorized posture** - Sitting, standing, or specific position
- **Authorized location** - Within radius of authorized coordinates
- **Required stillness** - Calm movement (<0.3 m/s)
- **Stillness duration** - Sustained for 3+ seconds

### Usage

```python
from governance_kernel.acorn_protocol import get_acorn_protocol, BiometricReading, PostureState, StillnessLevel

acorn = get_acorn_protocol()

# Register operator
acorn.register_operator(
    operator_id=\"DR_AMINA_HASSAN\",
    authorized_postures=[PostureState.SITTING],
    authorized_locations=[
        (0.0512, 40.3129, 100.0),  # Dadaab clinic, 100m radius
    ],
    required_stillness=StillnessLevel.CALM
)

# Authenticate with biometric reading
reading = BiometricReading(
    posture=PostureState.SITTING,
    location_lat=0.0512,
    location_lng=40.3129,
    movement_velocity=0.15,  # Calm
    timestamp=datetime.utcnow()
)

authenticated, reason = acorn.authenticate(\"DR_AMINA_HASSAN\", reading)

if authenticated:
    # Inject secret from Google Secret Manager
    secret = acorn.inject_secret(\"DR_AMINA_HASSAN\", \"GOLDEN_THREAD_KEY\", reading)
    print(f\"üîë Secret injected: {secret[:20]}...\")
else:
    print(f\"‚ùå Authentication failed: {reason}\")
```
