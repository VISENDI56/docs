## Three regulation modes

### ZEN Mode (High Anxiety >0.7)

**Goal:** Reduce cognitive load immediately. Summarize & pause.

```python
# Input: Dense technical report (500 words)
# Output: Executive summary + expandable details

"""
**Insight Summary:** The epidemiological vector analysis indicates 
a stochastic surge in R0 values...

*[System: Taking a beat. Complex details available below.]*

---> **Expand for Full Details (500 words)**
"""
```

**Features:**
- First sentence extraction
- Complexity warning
- Expandable full content
- No information loss

### FLOW Mode (Medium Anxiety 0.4-0.7)

**Goal:** Chunking. Don't remove info, but structure it for rhythm.

```python
# Input: Wall of text
# Output: Semantic bullet points

"""
â€¢ The epidemiological vector analysis indicates a stochastic 
  surge in R0 values correlated with spatiotemporal clustering.

â€¢ We advise immediate quarantine protocols involving negative 
  pressure isolation.

â€¢ Contact tracing heuristic applying Bayesian updates to 
  2nd degree nodes.
"""
```

**Features:**
- Semantic chunking (not arbitrary)
- Breathing units based on punctuation
- Maintains information density
- Improves scanability

### RAW Mode (Low Anxiety <0.4)

**Goal:** High bandwidth efficiency. User is calm and focused.

```python
# Input: Technical report
# Output: Unchanged

"""
The epidemiological vector analysis indicates a stochastic surge 
in R0 values correlated with the spatiotemporal clustering of 
sub-variants in Sector 7. We advise immediate quarantine protocols...
```

Silent Flux uses a lightweight **GaussianNB** classifier:

## Text entropy calculation

Approximates cognitive load of text:

```python
def calculate_text_entropy(text: str) -> float:
    """
    High score = Dense, jargon-heavy
    Low score = Simple, airy
    """
    words = re.findall(r'\w+', text)
    
    # Metrics
    avg_word_len = sum(len(w) for w in words) / len(words)
    sentences = re.split(r'[.!?]+', text)
    avg_sent_len = len(words) / max(1, len(sentences))
    
    # Heuristic formula
    return (avg_word_len * 0.5) + (avg_sent_len * 0.1)
```

## Semantic chunking

Breaks text into "breathing units":

```python
def chunk_text_semantically(text: str, max_chunk_size: int = 50) -> List[str]:
    """
    Based on punctuation and semantic completeness,
    not just character count.
    """
    sentences = re.split(r'(?<=[.!?]) +', text)
    chunks = []
    current_chunk = ""
    
    for sent in sentences:
        if len(current_chunk) + len(sent) < max_chunk_size * 5:
            current_chunk += sent + " "
        else:
            chunks.append(current_chunk.strip())
            current_chunk = sent + " "
    
    if current_chunk:
        chunks.append(current_chunk.strip())
    
    return chunks
```
