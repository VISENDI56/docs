## SovereignGuardrail

The core enforcement engine that validates all actions against compliance constraints.

### Basic usage

```python
from governance_kernel.vector_ledger import SovereignGuardrail, SovereigntyViolationError

guardrail = SovereignGuardrail()

# Validate a data transfer action
try:
    guardrail.validate_action(
        action_type='Data_Transfer',
        payload={
            'data_type': 'PHI',
            'destination': 'Foreign_Cloud'
        },
        jurisdiction='GDPR_EU'
    )
except SovereigntyViolationError as e:
    print(f"âŒ {e}")
    # Raises: "Violates GDPR Art. 9 (Processing of special categories)"
```

### Action types

| Action Type | Description | Required Fields |
|-------------|-------------|-----------------|
| `Data_Transfer` | Transfer of health data | `data_type`, `destination` |

## Tamper-proof audit trail

Every sovereignty decision is cryptographically logged with:

- **SHA-256 hash chain** - Immutable audit trail
- **Cloud KMS signature** - Non-repudiation
- **Bigtable storage** - High-throughput logging
- **Spanner sync** - Cross-region consistency

### Enable tamper-proof audit

```python
from governance_kernel.vector_ledger import SovereignGuardrail

# Initialize with tamper-proof audit
guardrail = SovereignGuardrail(enable_tamper_proof_audit=True)

# All actions are automatically logged
guardrail.validate_action(
    action_type='High_Risk_Inference',
    payload={
        'actor': 'ml_system',
        'resource': 'patient_diagnosis',
        'explanation': 'SHAP values: [0.8, 0.1, 0.1]',
        'confidence_score': 0.95,
        'evidence_chain': ['fever', 'cough', 'positive_test'],
        'consent_token': 'valid_token',
        'consent_scope': 'diagnosis'
    },
    jurisdiction='EU_AI_ACT'
)

# Retrieve audit history
history = guardrail.get_tamper_proof_audit_history(limit=10)

integrity = guardrail.verify_audit_chain_integrity()

```

## Humanitarian constraints

The Ethical Engine applies Geneva Convention and WHO IHR constraints with humanitarian margin calculations.

### Basic usage

```python
from governance_kernel.ethical_engine import EthicalEngine, HumanitarianViolationError

engine = EthicalEngine()

# Validate cholera outbreak response in refugee camp
try:
    result = engine.apply_constraints(
        action={
            'type': 'cholera_response',
            'scope': 'refugee_camp',
            'estimated_civilian_impact': 0.3,
            'medical_benefit': 0.85,
            'attack_rate': 0.04,
            'r_effective': 2.8,

            'conflict_zone': False,
            'outbreak_suspected': True,
```

## Crypto Shredder (IP-02)

Data is not deleted; it is cryptographically dissolved.

```python
from governance_kernel.crypto_shredder import CryptoShredder

shredder = CryptoShredder()

# Encrypt data with ephemeral key
encrypted_data, key_id = shredder.encrypt_with_ephemeral_key(
    data=patient_record,
    retention_days=180
)

# After retention period, shred the key
shredder.shred_key(key_id)

# Data is now cryptographically irrecoverable
# No deletion required - the key is gone
```
