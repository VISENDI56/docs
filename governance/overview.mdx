## Sectoral Compliance Engine

The Governance Kernel now includes a **Sectoral Abstraction Layer** that routes compliance checks based on operational context.

### Usage

```python
from governance_kernel.compliance_matrix import ComplianceMatrix, SectoralContext

matrix = ComplianceMatrix()

# Check supply chain compliance
result = matrix.check_sectoral_compliance(
    context=SectoralContext.SUPPLY_CHAIN,
    payload={
        "component_origin": "XUAR",
        "hardware_components": ["Tin", "Tantalum"]
    }
)

# Check humanitarian finance compliance (OFAC)
result = matrix.check_sectoral_compliance(
    context=SectoralContext.HUMANITARIAN_FINANCE,
    payload={
        "payment_initiation": True,
        "payee_id": "BENEFICIARY_001",
        "ofac_check_passed": False
    }

result = matrix.check_sectoral_compliance(
    context=SectoralContext.ESG_CARBON,
    payload={
        "goods_destination": "EU",
        "carbon_intensive": True,
        "embedded_emissions_calculated": False
    }
)
```

## SovereignGuardrail

The core enforcement engine that validates all actions against compliance constraints.

### Basic usage

```python
from governance_kernel.vector_ledger import SovereignGuardrail, SovereigntyViolationError

guardrail = SovereignGuardrail()

# Validate a data transfer action
try:
    guardrail.validate_action(
        action_type='Data_Transfer',
        payload={
            'data_type': 'PHI',
            'destination': 'Foreign_Cloud'
        },
        jurisdiction='GDPR_EU'
    )
except SovereigntyViolationError as e:
    print(f"âŒ {e}")
```

## Crypto Shredder (IP-02)

Data is not deleted; it is cryptographically dissolved.

```python
from governance_kernel.crypto_shredder import CryptoShredder

shredder = CryptoShredder()

# Encrypt data with ephemeral key
encrypted_data, key_id = shredder.encrypt_with_ephemeral_key(
    data=patient_record,
    retention_days=180
)

# After retention period, shred the key
shredder.shred_key(key_id)

# Data is now cryptographically irrecoverable
# No deletion required - the key is gone
```
