## Compliance enforcement

### SovereignGuardrail validation

Every action in iLuminara is validated against applicable frameworks:

```python
from governance_kernel.vector_ledger import SovereignGuardrail

guardrail = SovereignGuardrail()

# Example: KDPA Â§37 enforcement
try:
    guardrail.validate_action(
        action_type='Data_Transfer',
        payload={
            'data_type': 'PHI',
            'destination': 'Foreign_Cloud',
            'region': 'Kenya'
        },
        jurisdiction='KDPA_KE'
    )
except SovereigntyViolationError as e:
    print(f\"âŒ {e}\")
    # Output: \"Violates KDPA Â§37 (Transfer Restrictions)\"\n```

### Sectoral compliance routing

The Compliance Matrix routes checks based on operational context:

```python
from governance_kernel.compliance_matrix import ComplianceMatrix, SectoralContext

matrix = ComplianceMatrix()

# Healthcare data processing
result = matrix.check_sectoral_compliance(
    context=SectoralContext.DATA_PRIVACY,
    payload={
        'region': 'EU',
        'data_type': 'health_data',
        'processing_purpose': 'diagnosis'
    }
)

# AI inference
result = matrix.check_sectoral_compliance(
    context=SectoralContext.AI_GOVERNANCE,
    payload={
        'model_type': 'high_risk',
        'explanation_provided': True,
        'confidence_score': 0.95
    }
)
```
