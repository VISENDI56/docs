## Compliance frameworks

### GDPR (European Union)

**General Data Protection Regulation**

| Article | Requirement | iLuminara Implementation |
|---------|-------------|--------------------------|
| Art. 6 | Lawfulness of Processing | Consent validation via SovereignGuardrail |
| Art. 9 | Special Categories (Health Data) | PHI blocked from foreign transfers |
| Art. 17 | Right to Erasure | Crypto Shredder (IP-02) dissolution |
| Art. 22 | Right to Explanation | SHAP explainability for high-risk AI |
| Art. 30 | Records of Processing | Tamper-proof audit trail |
| Art. 32 | Security of Processing | CodeQL SAST, Gitleaks secret scanning |

**Enforcement:**
```python
guardrail.validate_action(
    action_type='Data_Transfer',
    payload={'data_type': 'PHI', 'destination': 'Foreign_Cloud'},
    jurisdiction='GDPR_EU'
)
# Raises: SovereigntyViolationError
```

**Kenya Data Protection Act**

| Section | Requirement | iLuminara Implementation |
|---------|-------------|--------------------------|

## Jurisdiction configuration

Configure your primary jurisdiction in `config/sovereign_guardrail.yaml`:

```yaml
jurisdiction:
  primary: "KDPA_KE"  # Kenya
  secondary:
    - "GDPR_EU"       # European Union
    - "POPIA_ZA"      # South Africa
    - "HIPAA_US"      # United States
```

## Compliance testing

### Test GDPR compliance

```python
import unittest
from governance_kernel.vector_ledger import SovereignGuardrail, SovereigntyViolationError

class TestGDPRCompliance(unittest.TestCase):
    def test_phi_cannot_leave_eu(self):
        guardrail = SovereignGuardrail()
        
        with self.assertRaises(SovereigntyViolationError):
            guardrail.validate_action(
                action_type='Data_Transfer',
                payload={
                    'data_type': 'PHI',
                    'destination': 'AWS_US'
                },
                jurisdiction='GDPR_EU'
            )
```

### Test HIPAA compliance

```python
def test_hipaa_audit_retention(self):
    from governance_kernel.crypto_shredder import CryptoShredder
    
    shredder = CryptoShredder()
    
    # Audit logs must be retained for 7 years
    assert shredder.audit_retention_days >= 2555
```

## Compliance reports

### Generate compliance report

```python
from governance_kernel.vector_ledger import SovereignGuardrail

guardrail = SovereignGuardrail()

# Generate compliance report
report = guardrail.generate_compliance_report(
    start_date='2025-01-01',
    end_date='2025-01-31',
    jurisdiction='KDPA_KE'
)

print(f"Total actions: {report['total_actions']}")
print(f"Violations: {report['violations']}")
print(f"Compliance rate: {report['compliance_rate']:.2%}")
```
