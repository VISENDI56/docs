## Compliance verification

### Check compliance status

```python
from governance_kernel.compliance_matrix import ComplianceMatrix

matrix = ComplianceMatrix()

# Get all enforced frameworks
frameworks = matrix.get_enforced_frameworks()
print(f"Enforced frameworks: {len(frameworks)}")

# Check specific framework
status = matrix.check_framework_compliance("GDPR_EU")
print(f"GDPR Status: {status['status']}")
print(f"Articles enforced: {len(status['articles'])}")

# Get compliance score
score = matrix.get_compliance_score()
print(f"Overall compliance: {score['percentage']:.1%}")
```
