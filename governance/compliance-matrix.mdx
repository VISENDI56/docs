## Usage examples

### Check compliance for specific framework

```python
from governance_kernel.compliance_frameworks import ComplianceMatrix, ComplianceFramework

matrix = ComplianceMatrix()

# Get GDPR requirements
gdpr_compliance = matrix.validate_compliance(ComplianceFramework.GDPR)

print(f"Framework: {gdpr_compliance['framework']}")
print(f"Total Requirements: {gdpr_compliance['total_requirements']}")
print(f"Mandatory: {gdpr_compliance['mandatory_requirements']}")

for req in gdpr_compliance['requirements']:
    print(f"  {req['article']}: {req['description']}")
    print(f"    Validation: {req['validation_method']}")
    print(f"    Penalty: {req['penalty']}")
```

### Get all mandatory requirements

```python
# Get all mandatory requirements across all 29 frameworks
mandatory = matrix.get_mandatory_requirements()

print(f"Total Mandatory Requirements: {len(mandatory)}")

for req in mandatory:
    print(f"âœ“ {req.framework.value}")
    print(f"  {req.article}: {req.description}")
    print(f"  Penalty: {req.penalty_for_violation}")
```

### Validate specific action

```python
from governance_kernel.vector_ledger import SovereignGuardrail

guardrail = SovereignGuardrail()

# Validate cross-border data transfer
try:
    guardrail.validate_action(
        action_type='Data_Transfer',
        payload={
            'data_type': 'PHI',
            'source_country': 'Kenya',
            'destination_country': 'USA',
            'consent_obtained': False
        },
        jurisdiction='KDPA_KE'
    )
except SovereigntyViolationError as e:
```
