### Basic usage

```python
from governance_kernel.omni_law_matrix import OmniLawMatrix, ComplianceContext

# Initialize matrix
matrix = OmniLawMatrix(enable_ai_triggers=True)

# Define operational context
context = ComplianceContext(
    action_type="prediction",
    data_type="PHI",
    jurisdiction="Kenya",
    ai_involved=True,
    high_risk=True,
    outbreak_context=True,
    metadata={
        "conformity_assessment": False,
        "who_notification": False
    }
)

# Validate action
result = matrix.validate_action(context)

print(f"Compliant: {result['compliant']}")
print(f"Activated Frameworks: {', '.join(result['activated_frameworks'])}")
print(f"Violations: {len(result['violations'])}")
```

## VI. AI & Digital Health Governance

The "Trustworthy Intelligence" stack for certifying FRENASA/ECF as high-risk AI.

### EU AI Act (Regulation 2024/1689)

World's first comprehensive AI framework. Health AI classified as high-risk.

**Requirements:**
- Conformity assessment before deployment
- Transparency documentation
- Post-market monitoring
- Human oversight

**iLuminara Implementation:**

```python
from governance_kernel.ai_governance import AIGovernance, AIRiskClass

governance = AIGovernance(enable_shap=True)

# Classify AI risk
risk_class = governance.classify_ai_risk(
    use_case="Cholera outbreak prediction",
    domain="health",
    impact="high"
)
# Result: AIRiskClass.HIGH

# Create conformity assessment
assessment = governance.create_conformity_assessment(
    model_id="FRENASA-ECF-v1",
    risk_class=risk_class,
    technical_documentation={
        "accuracy": {"train": 0.95, "test": 0.92},
        "robustness": {"adversarial_tested": True}
    }
)
```

### SHAP Explainability

## VII. Global Health Security & Outbreak Reporting

The "Pandemic Sentinel" stack for WHO integration.

### IHR 2005 (with 2025 amendments)

Binding WHO treaty requiring outbreak notification within 24 hours.

**iLuminara Implementation:**

```python
from governance_kernel.global_health_harmonizer import GlobalHealthHarmonizer, OutbreakEvent

harmonizer = GlobalHealthHarmonizer(
    national_focal_point="kenya.nfp@health.go.ke",
    country_code="KE",
    enable_auto_notification=True
)

# Create outbreak event
event = OutbreakEvent(
    event_id="KE-CHOLERA-2025-001",
    disease="cholera",
    location={"name": "Dadaab", "border_proximity": True},
    case_count=156,
    death_count=12,
    r_effective=2.8,
    severity_score=0.75
)

# Process event (auto-notifies WHO if required)
result = harmonizer.process_outbreak_event(event)
```
